# Plan: Add Dashboard Link to /chat Sidebar

## Problem
The `/chat` layout sidebar has no way to navigate to `/dashboard`. Users are stuck unless they manually edit the URL.

## Solution
Add a "Dashboard" icon button to the `SessionSidebar` header row, next to the existing `ThemeToggle` and `+` (new session) button.

## File to Modify
**`src/components/chat/SessionSidebar.tsx`**

### Changes

1. Add `useNavigate` import from `@tanstack/react-router` (already imports `useParams` from there — add to the same import).
2. Add `LayoutDashboard` (or `Home`) icon import from `lucide-react` (already imports `Plus`).
3. Inside the component, call `const navigate = useNavigate()`.
4. Insert a new icon button in the header `flex items-center gap-2` container — before `ThemeToggle` — that navigates to `/dashboard` on click.

### Button markup (to insert before `<ThemeToggle />`):
```tsx
<button
  type="button"
  onClick={() => navigate({ to: "/dashboard" })}
  className="flex min-h-[40px] min-w-[40px] items-center justify-center rounded-md border border-border/70 bg-background/70 text-muted-foreground hover:bg-muted hover:text-foreground press-scale"
  title="Dashboard"
  aria-label="Dashboard"
>
  <LayoutDashboard className="h-4 w-4" />
</button>
```
